# MajanScoreManager

麻雀の点数を管理するアプリです。

## 開発環境

- Xcode 12.4
- Swift 5.3.2


必要なパッケージのインストール

```sh
$ sudo gem install cocoapods
$ pod install --repo-update
```

finderから`MajanScoreManager.xcworkspace`を開く
